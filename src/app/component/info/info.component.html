Electronics Warehouse API

Project Purpose
      This project is created for learning purposes and to show the programming skills of back-end and front-end design in software development.
      Available functions are: create user and permissions group, add new item to the stock, and create sale order with managing and updating stock numbers accordingly.
    ---
    Technologies Used
        Back-End
      • Java 8.0
      • Spring Boot 2.4 (Data JPA Repository)
      • Log4J
      • JUnit
        Database
      • PostgreSQL 10.0
      • AWS RDS
        Front-End
      • Angular 11.0
      • Bootstrap 4.5 and CSS
        DevOps
      • Jenkins
      • AWS S3
